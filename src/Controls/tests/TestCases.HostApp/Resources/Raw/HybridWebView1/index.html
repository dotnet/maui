.NET MAUI HybridWebView

The HybridWebView control enables hosting web content in a WebView and facilitates communication between the code running in the WebView (JavaScript) and the code that hosts the WebView (C#/.NET). For example, if you have an existing React JS app, you could host it in a cross-platform .NET MAUI native app, and build the back-end of the app using C# and .NET.

DefaultFile
Type: string
Gets or sets the file within the HybridRoot that should be served as the default file. The default value is index.html.
HybridRoot
Type: string
Gets or sets the path within the app package that contains the web app's contents. The default value is wwwroot.

Events

RawMessageReceived
Type: EventHandler<HybridWebViewRawMessageReceivedEventArgs>
Occurs when a raw message is received from the JavaScript running in the WebView. The message is available in the Message property of the event args.
private void OnRawMessageReceived(object sender, HybridWebViewRawMessageReceivedEventArgs e) { string message = e.Message; // Process the message from JavaScript }

Methods

EvaluateJavaScriptAsync(string script)
Returns: Task<string>
Evaluates the JavaScript string in the context of the currently loaded page, if any.
var result = await hybridWebView.EvaluateJavaScriptAsync("document.title");

Usage Example

<HybridWebView x:Name="myHybridWebView" DefaultFile="index.html" HybridRoot="wwwroot" RawMessageReceived="OnRawMessageReceived" />
